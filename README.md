# Office Meeting Prep Agent

## Description
Agentic AI system that prepares users before meetings.

## Core Technologies/Tech Stack
Node.js  → Backend Agent Orchestrator
Microsoft Graph API → Outlook Calendar & Teams integration OR Mock Data Json
Semantic Kernel → Agent reasoning & memory
LangChain → LLM workflows & prompt orchestration
Azure OpenAI GPT-4o → AI reasoning & summarization

## High level architectural flow
User Input / Data (JSON) -> Node.js Backend (Agent Orchestrator) -> Semantic Kernel (Agent Logic) -> LLM (GPT-4o) -> Tools (Functions) -> Output (Console / Email)
